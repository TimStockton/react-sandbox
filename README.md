# react-sandbox

## Creating a React App

create-react-app is now deprecated...
Use Vite instead, which requires node.js t6o be installed, check with 'node -v' and 'npm -v'.

Create the Ract application:

```bash
npm create vite@latest . -- --template react
```

Install dependancies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
